SentimentIntensityAnalyzer() es una funcin que nos proporciona polarity_scores, que es un mtodo que facilita las puntuaciones para las siguientes categoras:

  • Negativo
  • Neutral
  • Positivo
  • Compuesto

La puntuacin compuesta es la suma de puntuaciones positivas, negativas y neutras, normalizadas entre -1 (negativo ms extremo) y +1 (positivo ms extremo).

def sent_score():

        return 'Negative'
    else :

3.3 Sentiment Analysis a partir del léxico con TextBlob:¶

  • El segundo enfoques para el anlisis de sentimiento se basa en el lxico y ea un modelo de de aprendizaje no supervisado. En los anlisis anteriores necesitamos datos preetiquetados, pero el segundo no, lo cual es una ventaja.
  • Textblob es una biblioteca utilizada para NLP (Natural Language Processing) que tiene diccionarios especiales (lxico) con informacin detallada, especialmente precisa y preparada para el anlisis de sentimientos. Estos lxicos tienen una lista de palabras polares positivas y negativas con una puntuacin asociada y usando varias tcnicas, como la posicin de las palabras, las palabras circundantes, el contexto, las partes del discurso, frases, etc., se asignan puntuaciones a los texto para el que queremos calcular el sentimiento, la suma total es el sentimiento final del texto.
In [263]:
from textblob import TextBlob
sent = TextBlob(text)
print(sent.sentiment)

Sentiment(polarity=0.02462876271015806, subjectivity=0.3963127957314004)

  • La polaridad es una mtrica utilizada para estimar el sentimiento general de un texto y vara de -1 a 1, donde -1 sera completamente negativo, 0 es neutral y 1 es completamente positivo. En este caso, est muy cerca de 0, por lo que el sentimiento general es neutral.

  • La subjetividad es una mtrica que va de 0 a 1, donde 0 es muy objetivo y 1 es muy subjetivo. En este caso est ms prximo a la objetividad, 0.39, al tratarse de una narracin donde el autor no tiene, evidentemente una opinion o posicin personal.
